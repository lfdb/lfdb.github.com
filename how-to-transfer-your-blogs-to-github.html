如何上传到Github

  • 注册一个主用户yache，安装好git并配置好ssh key，随时可以链接上
  • 注册子帐户，如yache@qq.com 和 qqz186@gmail.com 作为子帐户
  • 在子帐户里面新建一个Username.github.com repo，然后添加共同协作人yache（主帐户）
  • 接着在git bash里面按照如下步骤输入： 

 Set up git
  git config --global user.name "Your Name"
  git config --global user.email yache@qq.com
  Add your public key

 其实这一步可以省掉

Next steps:

  mkdir lfdb.github.com
  cd lfdb.github.com
  git init
  touch README
  git add README
  git commit -m 'first commit'
  git remote add origin git@github.com:lfdb/lfdb.github.com.git
  git push -u origin master

针对最后两句，把origin 改成对应的Username，即可完成上传

Existing Git Repo?

  cd existing_git_repo
  git remote add origin git@github.com:lfdb/lfdb.github.com.git
  git push -u origin master
